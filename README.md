# 🤖 CodeGuru – AI Code Teaching Assistant

CodeGuru is a **code teaching assistant** built using **Meta’s CodeLlama model** via **Ollama**, integrated with **Gradio** and **LangChain**.  
It answers programming questions, explains code, and helps developers learn interactively.

---

## 🚀 Features
- **[Code Assistant](ca://s?q=CodeGuru_code_assistant_features)**: Provides explanations and solutions to coding questions.
- **[Persistent Conversation](ca://s?q=CodeGuru_conversation_history)**: Maintains chat history for contextual responses.
- **[Custom System Prompt](ca://s?q=CodeGuru_system_prompt)**: Configured to act as a teaching assistant named *CodeGuru*.
- **[Gradio UI](ca://s?q=CodeGuru_Gradio_interface)**: Simple web interface for interaction.
- **[LangChain Integration](ca://s?q=LangChain_integration_with_CodeGuru)**: Enables chaining prompts and responses.

---

## 🛠️ Tech Stack
- **Python 3.10+**
- **Gradio** for UI
- **LangChain** for prompt chaining
- **Ollama** for running CodeLlama locally
- **Requests** for API calls
